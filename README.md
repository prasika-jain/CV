## 🌐 Live Demo

[🔗 View Website](https://prasika-jain.github.io/CV/) 

## 📌 Highlights

- Clean and professional UI with light color theme and rounded corners
- Sections include:
  - Contact Information
  - Professional Summary
  - Skills & Tools
  - Projects
  - Internship Experience
  - Education Details
  - Footer with GitHub and LinkedIn links
